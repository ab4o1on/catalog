# Doom для Лілки!

Це - порт класичної гри Doom для Лілки на ESP32-S3.

> **Doom** є власністю **id Software**. Цей проєкт є неофіційним портом і не пов'язаний з id Software.

## Особливості

- Модифікований рушій Doom Generic для запуску на ESP32-S3
- Динамічна алокація пам'яті для використання PSRAM
- Весь основний код Doom поміщається в 400 КБ основної RAM
- Підтримка звуку через I2S та п'єзодинамік

## Вимоги

Для того, щоб пограти в Doom, вам додатково потрібно мати файл `doom.wad`. Його можна купити або завантажити в інтернеті. Файл потрібно покласти в корінь SD-карти.
